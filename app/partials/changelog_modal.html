Telegram Web 升级记录

Telegram Web 已经升级完成！

当前版本 Version 0.1.3

  • 添加了您现在看到的欢迎页
  • 添加了键盘快捷键：
    • Tab - 将输入光标聚焦到撰写信息栏
    • Esc 或 Shift+Tab - 将输入光标聚焦到搜索栏
    • Up/Down (在搜索栏时) - 在不同聊天会话间切换
    • Enter (在搜索栏时) - 打开首个或选中的聊天会话
    • Alt+[0-9] - 切换到第 N 个聊天会话
    • Alt+Up/Down - 移动到 上一个/下一个 聊天会话
  • 改进了消息编组形式
  • 修复了视频附件模式的定位错误
  • 修复了移动端的光标聚焦错误
