Telegram Web 升级记录

Telegram Web 已经升级完成！

当前版本 Version 0.3.0

  • 登入验证码可以从其他 Telegram 客户端中获取
  • 添加了自动更新好友在线状态的功能
  • 修复了滚动条宽度不可用的错误
  • 可以在信息中使用非英语的 #标签
  • [Desktop] 添加了电话号码国家代码的自动判断
  • [FirefoxOS] 改进了推送提醒(版本 <= 1.1)
  • [FirefoxOS] 修复了表情符号在提醒中显示的问题
  • [FirefoxOS] 修复了附件错误(版本 <= 1.1)
  • [FirefoxOS] 添加了电话簿的权限处理
  • [FirefoxOS] 添加了分享相册照片到 Telegram 的功能

Version 0.2.9

  • 升级了移动布局
  • 修复一些问题

Version 0.2.5

  • 升级了移动布局
  • 性能有所提升
  • 改进了消息分组
  • [FirefoxOS] 修复了 <= 1.1 版本的提醒功能
  • [FirefoxOS] 修复了联系人导入功能

Version 0.2.1

  • 改进在慢速设备上密钥生成的速度
  • 添加移动设备的网络休眠支持
  • [FirefoxOS] 添加 PUSH 推送提醒支持

Version 0.2

  • 更新了联系人列表，添加编辑模式
  • 第一个 Firefox OS 发行版。详情请参照 Firefox Marketplace

Version 0.1.9

  • 提升性能
  • 修复一些错误

Version 0.1.8

  • 改进移动版布局
  • 修复一些错误

Version 0.1.7

  • 改进移动版布局
  • 添加Firefox/安卓文件缓存支持
  • 重新设计了转发消息功能
  • 支持 #Hash-tags
  • 修复消息传输的一些问题

Version 0.1.6

  • 添加聊天记录搜索功能
  • 改进未读消息过多聊天区性能
  • 修复一些其它问题

Version 0.1.5

  • 添加注册账号功能
  • 提升新用户的使用体验

Version 0.1.4

  • 查看用户头像历史（点击用户信息的头像即可显示）
  • 在设置中加入新的 GitHub, Twitter 按钮
  • 更新的键盘快捷键：
    • Alt+Home - 切换至最近一次的聊天回话
    • 移除了 Alt+1, 2, 3.. 快捷键，因为不同浏览器可能发生快捷键冲突

Version 0.1.3

  • 添加了您现在看到的欢迎页
  • 添加了键盘快捷键：
    • Tab - 将输入光标聚焦到撰写消息栏
    • Esc 或 Shift+Tab - 将输入光标聚焦到搜索栏
    • Up/Down (在搜索栏时) - 在不同聊天会话间切换
    • Enter (在搜索栏时) - 打开首个或选中的聊天会话
    • Alt+[0-9] - 切换到第 N 个聊天会话
    • Alt+Up/Down - 移动到 上一个/下一个 聊天会话
  • 改进了消息编组形式
  • 修复了视频附件模式的定位错误
  • 修复了移动端的光标聚焦错误
