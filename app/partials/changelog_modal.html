Telegram Web 升级记录

Telegram Web 已经升级完成！

当前版本 Version 0.1.8

  • 改进移动版布局
  • 修复一些错误

Version 0.1.7

  • 改进移动版布局
  • 添加Firefox/安卓文件缓存支持
  • 重新设计了转发信息功能
  • 支持 #Hash-tags
  • 修复信息传输的一些问题

Version 0.1.6

  • 添加聊天记录搜索功能
  • 改进未读信息过多聊天区性能
  • 修复一些其它问题

Version 0.1.5

  • 添加注册账号功能
  • 提升新用户的使用体验

Version 0.1.4

  • 查看用户头像历史（点击用户信息的头像即可显示）
  • 在设定中加入新的 GitHub, Twitter 按钮
  • 更新的键盘快捷键：
    • Alt+Home - 切换至最近一次的聊天回话
    • 移除了 Alt+1, 2, 3.. 快捷键，因为不同浏览器可能发生快捷键冲突

Version 0.1.3

  • 添加了您现在看到的欢迎页
  • 添加了键盘快捷键：
    • Tab - 将输入光标聚焦到撰写信息栏
    • Esc 或 Shift+Tab - 将输入光标聚焦到搜索栏
    • Up/Down (在搜索栏时) - 在不同聊天会话间切换
    • Enter (在搜索栏时) - 打开首个或选中的聊天会话
    • Alt+[0-9] - 切换到第 N 个聊天会话
    • Alt+Up/Down - 移动到 上一个/下一个 聊天会话
  • 改进了消息编组形式
  • 修复了视频附件模式的定位错误
  • 修复了移动端的光标聚焦错误
